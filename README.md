# AI RESUME ENHANCER 

AI Resume Enhancer is an intelligent tool designed to help job seekers optimize their resumes for Application Tracking Systems (ATS) and specific job descriptions. By leveraging Large Language Models (LLMs), this application parses resumes, compares them against job requirements, and provides actionable feedback and rewritten content.
